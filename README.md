# AI Personalized

An AI-based personalized recommendation system for e-learning.

# About

This system uses machine learning to recommend relevant courses and learning materials to students based on their interests, learning history, and behavior patterns.

# Features

- Personalized course recommendations
- User profile analysis
- Smart content matching
- Responsive web interface

# Tech Stack

- **Frontend**: HTML, CSS
